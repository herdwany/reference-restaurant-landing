# Client Access Policy

## العميل يحصل على

- رابط الموقع.
- رابط لوحة التحكم لاحقاً.
- حساب `owner` أو `staff`.
- صلاحية إدارة محتوى موقعه فقط.
- رؤية طلباته وحجوزاته فقط.

## العميل لا يحصل على

- Appwrite Console.
- GitHub repo.
- Vercel أو Netlify account.
- viaSocket account.
- API keys.
- Database direct access.
- صلاحية حذف المشروع الكامل.
- أدوات الوكالة الداخلية.

## نقل المشروع

إذا أراد العميل نقل المشروع:

- يمكن تصدير بياناته JSON/CSV حسب الاتفاق.
- يمكن تسليم نسخة من المحتوى والصور.
- لا يتم تسليم أدوات الوكالة الداخلية إلا باتفاق منفصل.
- أي نقل يجب أن يحافظ على خصوصية بيانات العملاء الآخرين.
