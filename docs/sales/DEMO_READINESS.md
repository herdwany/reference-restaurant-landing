# Demo Readiness

Checklist عملي قبل عرض Pixel One Visuals على صاحب مطعم أو كافيه.

## البيئة التقنية

- [ ] Appwrite Project يعمل بدون أخطاء.
- [ ] Appwrite Functions منشورة:
  - [ ] `createOrder`
  - [ ] `createReservation`
  - [ ] `trackRequest`
  - [ ] `createClient` إذا كان عرض الوكالة مطلوبًا.
- [ ] ملف `.env.local` مضبوط بالقيم الصحيحة.
- [ ] `VITE_APPWRITE_DATABASE_ID` و`VITE_APPWRITE_BUCKET_ID` صحيحان.
- [ ] `npm run build` ينجح قبل العرض.

## مطعم الديمو

- [ ] مطعم demo حالته `active`.
- [ ] رابط `/r/:slug` يفتح الموقع العام.
- [ ] لا توجد رسائل setup أو أخطاء ظاهرة للعميل.
- [ ] اسم المطعم، الوصف، العنوان، أوقات العمل، ورقم WhatsApp جاهزة.
- [ ] لا يوجد placeholder content ظاهر.

## المحتوى

- [ ] أطباق كافية للعرض، مع أسعار وصور أو صور بديلة جيدة.
- [ ] عرض واحد على الأقل جاهز إذا كانت العروض ضمن الباقة.
- [ ] صور المعرض جاهزة إذا كان المعرض ضمن الباقة.
- [ ] FAQ جاهزة إذا كانت ضمن العرض.
- [ ] الصور لا تبدو مكسورة أو بطيئة جدًا.

## الطلبات والحجوزات

- [ ] إرسال طلب تجريبي من الموقع العام يعمل.
- [ ] الطلب يظهر في `/admin/orders`.
- [ ] صفحة تتبع الطلب تعمل من `/r/:slug/track`.
- [ ] إرسال حجز تجريبي يعمل إذا كانت الحجوزات مفعلة.
- [ ] الحجز يظهر في `/admin/reservations`.
- [ ] تتبع الحجز يعمل من `/r/:slug/track`.

## اللغات والتجاوب

- [ ] العربية `ar` checked.
- [ ] الفرنسية `fr` checked.
- [ ] الإنجليزية `en` checked.
- [ ] اتجاه RTL في العربية سليم.
- [ ] اتجاه LTR في الفرنسية والإنجليزية سليم.
- [ ] الهاتف checked، خصوصًا المنيو، السلة، الحجز، وsidebar في `/admin`.
- [ ] لا يوجد horizontal scroll ظاهر.

## تسجيل الدخول

- [ ] owner login checked على `/admin`.
- [ ] staff login checked إذا كان مطلوبًا في العرض.
- [ ] agency login checked على `/agency`.
- [ ] صلاحيات الباقة تظهر بشكل صحيح، بدون كسر feature gates.
- [ ] زر الخروج يعمل.

## قبل الاجتماع مباشرة

- [ ] افتح الموقع العام في tab جاهز.
- [ ] افتح `/admin` في tab جاهز.
- [ ] افتح `/agency` فقط إذا كان العميل يحتاج شرح إدارة الوكالة.
- [ ] حضّر سيناريو: عرض الموقع، إرسال طلب، تتبع الطلب، تعديل طبق أو عرض.
- [ ] لا تعرض بيانات عميل حقيقي أثناء demo.
